# Travel Radar LK — Frontend Project

Travel Radar LK — контентный туристический проект, построенный на чистом фронтенде
с приоритетом на простоту, прозрачность архитектуры и долгосрочную поддержку.

Проект ориентирован на SEO, ручное управление контентом и отказ от избыточных технологий.

====================================================================

ОБЩАЯ АРХИТЕКТУРА

Проект реализован без фреймворков и сборщиков.

Используемый стек:
- HTML — без шаблонизаторов
- CSS — без фреймворков
- JavaScript — только для:
  - загрузки header / footer
  - навигации
  - минимальных UI-задач

НЕ используется:
- React / Vue
- Webpack / Vite
- Server-side rendering

====================================================================

PARTIALS (HEADER / FOOTER)

Каталог /Partials/ содержит HTML-фрагменты, которые динамически
подключаются ко всем активным страницам проекта.

АКТИВНАЯ СИСТЕМА (NEW)

Используется единая система загрузки partials через JavaScript.

ФАЙЛЫ:

- header.html
  - Шапка сайта
  - Основная навигация
  - Переключение языков (RU / UA / EN)
  - Переключение темы
  - Социальные ссылки

- footer.html
  - Подвал сайта
  - Навигация
  - Copyright

ПОДКЛЮЧЕНИЕ НА СТРАНИЦАХ:

Каждая активная HTML-страница содержит:

<div id="site-header"></div>

<!-- page content -->

<div id="site-footer"></div>
<script src="/js/load-header.js"></script>

КАК ЭТО РАБОТАЕТ:

1. Страница загружается
2. load-header.js выполняет fetch
3. HTML partials вставляются в DOM

fetch('/Partials/header.html')
fetch('/Partials/footer.html')
