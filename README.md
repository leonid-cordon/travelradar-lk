# travelradar-lk
Travel Radar LK — one-page travel landing page (Egypt winter tips) with quick checklists + links to Booking, flights, insurance &amp; eSIM. MVP for affiliate conversions.
**Travel Radar LK** — одностраничный travel-лендинг (MVP) со шпаргалками по **Египту зимой** и быстрыми ссылками на бронирование отелей, билеты, страховку и eSIM.

## Что это

Этот репозиторий содержит **одну страницу (index.html)**, которая:
- даёт короткие чек-листы и “шпаргалки” (ветер, бухты, красный флаг, план Б);
- ведёт на полезные сервисы для поездки;
- используется как простой сайт-визитка для теста конверсий (MVP).
